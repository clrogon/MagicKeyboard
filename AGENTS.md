# Agent Configuration: MagicKeyboard

Welcome to the MagicKeyboard repository. This file provides context and permissions for AI coding agents.

## Project Context
- **Name:** MagicKeyboard
- **Tech Stack:** TypeScript, Vite
- **Architecture:** Single-package

## Permissions & Constraints

### Allowed Actions
- Run tests
- Run lint
- Read all files

### Forbidden Actions
- **DO NOT** push directly to the `main` branch.
- **DO NOT** modify, create, or delete any secrets or environment variables.

## Getting Started
Agents should begin by indexing the codebase and checking the `.agents/rules/` directory for specific guidelines.
